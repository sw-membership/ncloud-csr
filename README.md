# ncloud-csr

네이버 클라우드 플랫폼 CSR(Clova Speech Recognition) API 사용을 위한 코드입니다.

## Getting Started

다음을 설치해야 합니다.

- [ffmpeg](https://www.ffmpeg.org/)

- [libab](https://libav.org/)

- requirements.txt

이후 현재 디렉토리에 `audio.mp3` 파일을 넣고 `python3 main.py` 를 실행하시기 바랍니다.
